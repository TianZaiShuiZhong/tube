#### **1. 直管（PIPE288）算例**

图 1 直管 PIPE288 算例示意（示意图）

**弹塑性材料参数**：

| 参数项           | 值                     |
| ---------------- | ---------------------- |

| 泊松比           | 0.3                    |
| 密度             | 7800 kg/m³             |
| 热膨胀系数       | 1.2E-5 (参考温度 25°C) |
| BISO模型屈服应力 | 25 MPa                 |
| 切线模量         | 100 GPa                |

**蠕变材料参数**

| 参数项                                 | 值                     |

| Creep (各向同性 Strain Hardening) - C4 | 0                      |

**单元与截面设置**

- **PIPE288 单元特性设置：**

**Options for PIPE288, Element Type Ref. No. 1**

Temperature input gradient (K1): Through wall

Element behavior (K3): Linear Form

Hoop strain treatment (K4): Thin shell

Internal / external pressures (K6): End cap loads

Section force/strain output (K7): At intgr points

Shear stress output (K8): Combined State

Stress / Strain (sect points) (K9): NONE

Extrapolated (elmt/sect nds) (K11): NONE

Hydronamic output (K12): NONE

Results file format (K15): Avg (corner nds)



- **PIPE288 截面参数设置：** 图 3 PIPE288 单元截面设置（示意图）

**Add or Edit Pipe Section** (添加或编辑管道截面)

**各部分内容**

**[SECTYPE] 截面类型**

**Edit Pipe Section 1** (编辑管道截面 1)

**Section Name** (截面名称): `STRAI`

**[SECDATA] 截面数据**

**Pipe diameter** (管道直径): `30`

**Wall thickness** (壁厚): `1`

**Circumferential divisions** (圆周方向划分份数): `20`

**Composite shell ID** (复合壳ID): `None`(无)

**Radial divisions (non-comp)** (径向划分份数-非复合): `5`

**Internal fluid material** (内部流体材料): `None`(无)

**Insulation material** (保温层材料): `None`(无)

**Insulation thickness** (保温层厚度): `0`

**[SECOFFSET] 截面偏移**

**Offset node to** (节点偏移至): `0`

**[SECCONTROL] 截面控制**

**Added Mass (Mass/Length)** (附加质量 [质量/长度]): `0`

**外载荷**

1. **自重：** 方向为 -Z，大小为 9.8 m/s²。
2. **均布内压：** 大小为 3 MPa。
3. **均匀温升：** 温度 200°C。

**算例建模文件**

```
PIPE288.cdb
```

**核对说明**

- 已核对：CDB 文件中 `SECDATA` 行的截面参数与本说明一致，数值单位为毫米（mm）：外径 = 30 mm，壁厚 = 1 mm。解析器按该单位直接使用这些数值计算截面面积与惯性矩，若希望使用米（m）为单位，请在输入和文档中统一换算（1 m = 1000 mm）。
